# Раки на замовлення

Простий статичний сайт на HTML, CSS та JavaScript.

## Файли

- `index.html` — головна сторінка з розділами про раків
- `styles.css` — стилі сторінки

## Запуск

Відкрийте `index.html` у браузері.

## Зміна

Редагуйте текст та секції в `index.html`, а оформлення — в `styles.css`.
